Фарминг в travian

Фарм - похищение ресурсов у других игроков с помощью войск.

Необходимо:
- войска
- кормушка
- регулярная отправка войск

Библиотека travlib реализует api для определения количества войск в деревне,
отправку набегов, просмотр логов.
Библиотека botlib определяет классы для выполнения полного цикла фарма.

Функционал botlib:
- Сканирование карты
    - Определение деревень Натар
    - Определение малоактивных игроков
    - Определение оазисов
- Анализ целей для фарма
    - Анализ населения
    - Разведка войск и сооружений
    - Просмотр логов
- Похищение ресурсов
    - Отправка набегов
    - Зачистка оазисов
- Оптимизация
    - Ожидание прибытия войск
    - Изменение размеров отрядов-грабителей
